Terrain
=======

Agrega la informacion de altura (coordenada Z) a un geoJSON utilizando como basa un archivo DEM.

Instalar los requerimientos 
```npm install```

Ejecutar tests
```npm run tests```

Ejecutar servidor en el puerto 3004
```npm run start```
